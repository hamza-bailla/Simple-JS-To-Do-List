#  Simple Vanilla JS To-Do List

A straightforward To-Do List application created to practice foundational web development skills using HTML, CSS, and pure JavaScript (Vanilla JS).

This project focuses on **DOM manipulation**, **event handling**, and **clean code structure** without the need for external frameworks or libraries.

##  Features

* **Add Tasks:** Easily add new tasks via the input field.
* **Mark as Done:** Click the "done" button to toggle the completion status, visually strikethrough the task.
* **Input Validation:** Alerts the user if they try to add an empty task.
* **Responsive Design:** Styled with simple CSS for a clean, readable interface.

##  How to Run Locally

1.  **Clone the Repository:**
    ```bash
    git clone [YOUR_REPO_URL]
    cd simple-todo-list
    ```

2.  **Open the File:**
    Simply open the `index.html` file in your web browser. There are no server dependencies required.
